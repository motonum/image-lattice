# Image Lattice

簡易的なブラウザベースの画像グリッドツール（Vite + React + TypeScript）

目的: 行数・列数を指定してグリッドを作り、各セルに画像を追加・ラベル編集し、ギャップを指定して1枚の PNG としてダウンロードできます。

クイックスタート:

1. 依存関係をインストール

   npm install

2. 開発サーバを起動

   npm run dev

注意: TIF サポートには `tiff.js` を使っています。環境によってはインストールされている package 名が異なる場合があります。

簡単な使い方:

- Rows/Cols でグリッドを生成
- セルに画像をドラッグ＆ドロップ、またはファイル選択
- 画像をドラッグしてセルを移動（空セル: 単純移動、占有セル: 挿入/ずらし）
- Download PNG で最終出力
